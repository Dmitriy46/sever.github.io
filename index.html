📁 Синхронизация файлами

📱 Ручная синхронизация: Экспорт/импорт файлов для переноса данных между устройствами
💡 Как работает:
1. Экспорт - сохраняет файл с данными
2. На другом устройстве откройте сайт
3. Импорт - загрузите файл
4. Данные появятся на новом устройстве

🤖 Настройки Telegram

📋 Как получить данные:
1. Напишите @BotFather в Telegram
2. Отправьте /newbot
3. Следуйте инструкциям и получите токен
4. Добавьте бота в группу/напишите ему
5. Отправьте любое сообщение
6. Откройте: api.telegram.org/bot{ТОКЕН}/getUpdates
7. Скопируйте chat_id из ответа
🔒 Безопасность: Данные сохраняются только в вашем браузере и не передаются никому.

🔥 Настройка Firebase

📋 Пошаговая инструкция:
1. Откройте console.firebase.google.com
2. Создайте новый проект
3. Database → Realtime Database → Создать
4. Настройки проекта → Добавить приложение → Web
5. Скопируйте настройки сюда

🐙 Настройка GitHub Gist

📋 Пошаговая инструкция:
1. Откройте github.com/settings/tokens
2. Generate new token → Fine-grained tokens
3. Выберите срок действия (например, 1 год)
4. Repository access: All repositories
5. Permissions: Contents (Read and write), Metadata (Read)
6. Generate token и скопируйте сюда
Токен нужен для создания и обновления Gist с вашими данными
Если у вас уже есть Gist с данными, вставьте его ID
Рекомендуется: данные будут видны только вам
💡 Преимущества GitHub Gist:
• История изменений (версионность)
• Надежность GitHub
• Возможность просмотра в браузере
• Автоматическое резервное копирование
